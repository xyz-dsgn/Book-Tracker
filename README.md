## Описание программы 
*"Программа для управления списком прочитанных книг. Позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять и загружать данные в формате JSON.
## Требования для запуска 
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: `pip install tkinter` --- 
## Как запустить 
```bash 
git clone https://github.com/xyz-dsgn/Book-Tracker
cd Book-Tracker 
python main.py 
Как пользоваться 
Введите данные книги и нажмите «Добавить книгу».
Используйте фильтры для поиска по жанру или минимальному количеству страниц.
Сохраните список в JSON-файл или загрузите ранее сохранённый.
Пример работы 
1.Вы запускаете программу и видите окно с полями для добавления книги и таблицей.
2.Вводите новую книгу:
•Название книги: «Обитаемый остров»
•Автор: «Федор Достоевский»
•Жанр: «Драма»
•Страниц: 350
3.Нажимаете «Добавить книгу» — книга появляется в списке.
4.В поле «Фильтр по жанру» вводите «Драма» и нажимаете «Применить фильтр» — в таблице показываются только книги жанра «Драма».
5.В поле «Мин. страниц» вводите «300» и нажимаете «Применить фильтр» — осталась только «Обитаемый остров».
6.Чтобы увидеть все книги, нажимаете «Сбросить фильтр».
7.Выбираете «Файл» → «Сохранить в JSON», сохраняете файл.
8.В следующий раз, если вы откроете программу и выберете «Файл» → «Загрузить из JSON», все ваши книги снова появятся.
